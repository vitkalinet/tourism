## ✏️ Редактирование экскурсий

Каждая экскурсия — отдельный JSON-файл в `data/tours/`.

### Как добавить новую экскурсию:

1. Создайте файл `data/tours/new-tour.json` (имя = slug)
2. Скопируйте структуру из любого существующего файла
3. Заполните все поля
4. Добавьте импорт в `src/data/tours.ts`
5. Перезапустите dev-сервер

Подробнее: [data/README.md](data/README.md)

## 📱 Функционал

### Главная страница

- **Hero** — полноэкранный блок с фото, статистикой, CTA
- **О гиде** — фото, биография, аккредитации
- **Преимущества** — 6 карточек
- **Каталог** — 12 экскурсий с фильтрацией по категориям
- **Авто-туры** — выездные туры
- **Стоимость** — 3 тарифа
- **Контакты** — Max, Telegram, форма заявки
- **Плавающая кнопка Max** — видна на всех страницах

### Страница экскурсии

- **Hero** — большое фото, название
- **Программа маршрута** — таймлайн по времени
- **Что включено / Не включено** — чек-листы
- **Важная информация** — длительность, группа, одежда, сезон
- **CTA-блоки** — кнопки "Забронировать" и "Задать вопрос" (Max)
- **Вам также может понравиться** — связанные экскурсии

## 🔗 Контакты

Все кнопки ведут в Max с предзаполненным сообщением:

> «Здравствуйте, Зурган! Хочу забронировать экскурсию [Название] на [Дата]. Количество человек: [N].»

Номер Max: `79001234567`
