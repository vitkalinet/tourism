# Зурган — Ваш гид по Калмыкии

Эксклюзивные экскурсии и авторские туры по Республике Калмыкия — единственному региону в Европе с буддийской культурой и кочевыми традициями.

## 🌟 Экскурсии

- **Буддизм легко и доступно** — экскурсия по хурулам (Золотая обитель Будды Шакьямуни)
- **Прогулка по Элисте** — пешая экскурсия от Пагоды семи дней до парка «Дружба»
- **Одинокий тополь и родники** — место силы с целебными источниками
- **Музей Пальмова** — Национальный музей с экспозицией «Степь как жизненное пространство»
- **Поющие барханы и сайгаки** — заповедная зона с антилопами и розовым озером
- **Кочевники и верблюды** — поход на верблюдах и чаепитие в юрте
- **Конные прогулки** — оздоровительные прогулки с пикником
- **Цветение лотосов** — речная прогулка к храму Майтреи
- **Тибетская медицина и астрология** — оздоровительные программы
- **Массаж и отдых** — классический, тибетский, баночный массаж
- **Ночевка в юрте** — погружение в быт кочевника
- **Шахматы и Остап Бендер** — шахматная столица России

## 🛠 Технологии

- **Next.js 16** — React-фреймворк с App Router
- **TypeScript** — типизация кода
- **Tailwind CSS 4** — утилитарные стили
- **SCSS Modules** — модульные стили компонентов
- **Anime.js** — анимации (для legacy-компонентов)

## 🚀 Запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен-сервера
npm run start
```

```
src/

│   ├── page.tsx             # Главная страница
│   └── tour/
│       └── [slug]/

│           ├── not-found.tsx # Страница 404
│           └── TourPage.module.scss

│   ├── Navigation.tsx       # Навигационное меню (мобильное)
│   ├── Navigation.module.scss
│   ├── HeroSection.tsx      # Hero-блок с анимациями
│   ├── HeroSection.module.scss
│   ├── AboutGuide.tsx       # Блок о гиде Зургане
│   ├── AboutGuide.module.scss

│   ├── Advantages.module.scss
│   ├── TourCards.tsx        # Каталог экскурсий с фильтрами
│   ├── TourCards.module.scss
│   ├── CarTours.tsx         # Автомобильные туры
│   ├── CarTours.module.scss
│   ├── Pricing.tsx          # Тарифы и цены
│   ├── Pricing.module.scss

```

## 🎨 Дизайн

- **Цветовая палитра**: тёмный фон, золотые акценты, песочные и бирюзовые оттенки
- **Шрифты**: Cormorant Garamond (заголовки), Geologica (текст), Unbounded (акценты)
- **Анимации**: плавные появления при скролле (Intersection Observer)
